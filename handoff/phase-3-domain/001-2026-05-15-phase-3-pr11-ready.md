## Open questions / risks

- **`datetime.utcnow()` deprecation warnings.** `ActivityBucket`'s
  `bucket_start` default uses `datetime.utcnow`, which is deprecated in
  Python 3.12 (the project's upper-bound matrix). Tests currently pass
  cleanly because the call sites use a fixed `NOW` constant. If Phase 4
  introduces real-time clock calls into the domain layer, switch to
  `datetime.now(UTC)` and audit downstream timestamp comparisons (most
  domain code currently assumes naive UTC).

- **`docs/spec/original-skeleton.md` rename interaction with Phase 4.**
  PR #10 moves the spec file. If Phase 4 starts before #10 merges, the
