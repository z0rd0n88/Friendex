Manual TDD — `/tdd` slash command not available. Workflow per AC:
1. Author the failing test.
2. Run `uv run pytest <node>` and capture stderr/stdout verbatim.
3. Implement the minimal code to make it pass.
4. Run again, capture GREEN, move on.

- **`/balance` with snapshot=None:** plan to reply ephemerally with a short
  "No account found — run /daily to open one" message via a tiny
  `discord.Embed` constructed inline (no new public builder). Keeps the
  account_cog's only branch trivial and doesn't dirty the embed builder
  surface. Revisit if this drives test churn.
- **`setup()` vs `register()` entry point:** plan to expose neither for now —
  Phase 13/14 will wire cogs via container code, not `bot.load_extension`.

  `command.default_permissions` (preferred over walking `checks` since

  `app_commands.Command`).
