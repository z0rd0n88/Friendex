## Overview

Friendex is a Python Discord bot that simulates a per-guild stock exchange: each server member has a tradeable stock whose price moves in response to real Discord activity (messages, voice time, reactions). It is a greenfield hexagonal (ports-and-adapters) rebuild under `src/friendex/`, managed with `uv` and targeting Python 3.11+. The architecture flows inward: `adapters/` (discord_bot, persistence via async SQLAlchemy + SQLite, config via pydantic-settings) вҶ’ `application/` (use-case services) вҶ’ `domain/` (pure dataclass models and error taxonomy, no external imports). The entry point will be `friendex.main:main` (wired in a later phase); until then the exercisable surface is the test suite (`uv run pytest`). CI runs ruff, mypy --strict, and pytest across Python 3.11 and 3.12 on every push and PR.
